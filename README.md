# Mini-Humanoid-Atlas 🧦🤖

**A fork focused on small-scale (1ft / ~30cm) humanoid / android projects**  
Built on top of [humanoid-atlas](https://github.com/kingjulio8238/humanoid-atlas) + [PR #8 seed-agi alignment](https://github.com/kingjulio8238/humanoid-atlas/pull/8)

## 🚀 Today's Highlight — March 28 2026
**Unitree UnifoLM-WBT-Dataset** (released March 5 2026, with rolling updates)

High-quality real-world **whole-body teleoperation (WBT)** dataset for humanoids in open environments.  
Includes tasks like clothing manipulation — ideal for training small hopping androids that fold laundry!

**Links:**
- Hugging Face collection: https://huggingface.co/collections/unitreerobotics/unifolm-wbt-dataset
- Unitree Robotics: https://www.unitree.com
